# 飞书文档操作 Skill

通过飞书 MCP Server 实现对飞书文档、知识库、电子表格、多维表格和消息的全面操作。

## 参考文件

| 文件 | 用途 |
| ---- | ---- |
| `references/setup-guide.md` | 飞书 MCP Server 安装配置指南（3 种方案） |
| `references/tools-reference.md` | 飞书 MCP 工具完整参考（文档/知识库/表格/消息/任务） |

## 工作流

### 检测飞书 MCP 可用性

每次执行前先检测飞书 MCP 工具是否可用：

1. 搜索可用工具中包含 `feishu`、`lark`、`飞书` 的 MCP 工具
2. **如果找到**：记录可用工具列表，进入对应操作工作流
3. **如果未找到**：提示用户需要配置飞书 MCP Server，加载 `references/setup-guide.md` 引导配置

### 查询/读取文档

当用户需要读取飞书文档内容时：

1. **解析输入** — 从用户输入中提取文档 ID 或 URL
   - URL 格式：`https://{domain}.feishu.cn/docx/{document_id}`
   - 直接提供文档 ID
2. **加载工具参考** — 读取 `references/tools-reference.md` 确认工具名称和参数
3. **调用 MCP 工具** — 使用文档读取工具获取内容
4. **格式化输出** — 将文档内容整理为可读的 Markdown 格式
5. **补充元信息** — 提供文档标题、最后更新时间等上下文

### 搜索文档

当用户需要搜索飞书文档时：

1. **提取搜索关键词** — 从用户描述中提炼搜索词
2. **调用搜索工具** — 使用文档搜索 MCP 工具
3. **整理结果** — 以表格形式展示搜索结果（标题、类型、链接、更新时间）
4. **按需深入** — 如用户需要，进一步读取具体文档内容

### 浏览知识库

当用户需要浏览飞书知识库时：

1. **获取知识空间** — 列出可用的知识空间
2. **浏览节点树** — 展示知识库的目录结构
3. **读取节点内容** — 根据用户选择读取具体页面内容
4. **关联导航** — 提示相关的子页面或关联文档

### 操作电子表格

当用户需要读写飞书电子表格时：

1. **解析表格信息** — 提取 spreadsheet_token 和 sheet_id
2. **读取数据** — 获取指定范围的单元格数据
3. **数据处理** — 按用户需求分析、筛选或汇总数据
4. **写入数据**（如需要）— 将处理结果写回表格

### 操作多维表格

当用户需要操作飞书多维表格时：

1. **解析表格信息** — 提取 app_token 和 table_id
2. **查询记录** — 使用过滤条件获取目标记录
3. **数据操作** — 创建 / 更新 / 删除记录
4. **结果确认** — 展示操作结果和影响行数

### 创建/编辑文档

当用户需要创建或编辑飞书文档时：

1. **确认目标** — 新建文档还是编辑已有文档
2. **准备内容** — 将用户提供的内容转换为飞书文档块格式
3. **执行操作** — 调用创建/更新文档块工具
4. **验证结果** — 重新读取文档确认修改生效

### 发送消息

当用户需要通过飞书发送消息时：

1. **确认目标** — 确定接收人或群聊
2. **构造消息** — 根据消息类型（文本/富文本/卡片）组织内容
3. **发送** — 调用消息发送工具
4. **确认** — 展示发送结果

### 配置引导

当飞书 MCP 不可用或用户需要配置时：

1. **加载配置指南** — 读取 `references/setup-guide.md`
2. **推荐方案** — 根据用户需求推荐合适的 MCP Server 方案
3. **逐步引导** — 按步骤指导完成配置
4. **验证配置** — 引导用户测试 MCP 连接

## 文档 ID 提取规则

```text
飞书 URL → Token 映射：
┌────────────────────────────────────────┬──────────────────┐
│ URL 路径                               │ Token 类型       │
├────────────────────────────────────────┼──────────────────┤
│ /docx/{id}                             │ document_id      │
│ /wiki/{id}                             │ wiki_token       │
│ /sheets/{id}                           │ spreadsheet_token│
│ /base/{id}                             │ app_token        │
│ /drive/folder/{id}                     │ folder_token     │
└────────────────────────────────────────┴──────────────────┘
```
